# 학식 연동 AI 영양분석 앱 - Frontend

이 프로젝트는 Flutter로 구현하는 학식 연동 AI 영양분석 앱의 프론트엔드입니다.

## 주요 기능

- 여러 식당의 학식 메뉴 조회
- 날짜별 학식 메뉴 조회
- 메뉴 상세 정보 확인
- AI 영양분석 결과 표시
- 먹을 메뉴 선택
- 선택 메뉴 영양정보 합산
- 1일 권장량 대비 비교 표시

## 먼저 읽을 문서

처음 작업하는 팀원은 아래 순서대로 문서를 읽습니다.

1. `docs/01_VIBE_CODING_GUIDE.md`
2. `docs/02_GIT_BRANCH_GUIDE.md`
3. `docs/03_FRONTEND_GUIDE.md`
4. 본인에게 해당하는 TASK 문서

준영 님:

- `docs/04_TASK_준영_MENU_LIST.md`

승헌 님:

- `docs/05_TASK_승헌_NUTRITION_SUMMARY.md`

## 작업 방식

이 프로젝트는 AI 코딩 에이전트를 활용해 개발합니다.

## 브랜치 규칙

`main` 브랜치에서 직접 작업하지 않습니다.

각자 작업 브랜치를 만들어 작업합니다.

예시:

- 준영 님: `feature/menu-list-screen`
- 승헌 님: `feature/nutrition-summary-screen`
