# 준영 님 작업 가이드: 학식 조회 화면

## 1. 이번 작업의 목적

이번 작업은 사용자가 앱을 열었을 때 가장 먼저 보게 되는 “오늘의 학식 화면”을 만드는 것입니다.

이 화면에서는 학교 안의 여러 식당 메뉴를 식당별로 묶어서 보여줍니다.

사용자는 여기서 메뉴를 확인하고, 먹을 메뉴를 체크하거나, 메뉴 상세 화면으로 이동할 수 있어야 합니다.

---

## 2. 이 작업을 통해 배우는 것

이번 작업을 하면서 다음을 익히는 것이 목표입니다.

```text
- Flutter에서 화면을 구성하는 방법
- 리스트 데이터를 카드 UI로 반복해서 표시하는 방법
- 위젯을 작게 나누는 방법
- 화면 이동을 연결하는 방법
- AI 코딩 에이전트에게 작은 단위로 요청하는 방법
- Git 브랜치를 만들어 안전하게 작업하는 방법
```

---

## 3. 작업 전에 읽을 문서

작업 전에 아래 문서를 읽습니다.

```text
1. docs/01_VIBE_CODING_GUIDE.md
2. docs/02_GIT_BRANCH_GUIDE.md
3. docs/03_FRONTEND_GUIDE.md
```

---

## 4. 작업 브랜치 만들기

main 브랜치에서 직접 작업하지 않습니다.

아래 명령어로 작업 브랜치를 만듭니다.

```bash
git switch main
git pull origin main
git switch -c feature/menu-list-screen
```

이미 브랜치를 만들어둔 상태라면 다음 명령어로 이동합니다.

```bash
git switch feature/menu-list-screen
```

---

## 5. 담당 기능

준영 님이 담당하는 기능은 다음입니다.

```text
- 오늘의 학식 화면
- 날짜별 메뉴 화면
- 식당별 메뉴 목록 화면
- 메뉴 카드 UI
- 메뉴 상세 화면으로 이동하는 흐름
```

---

## 6. 작업 순서

아래 순서대로 작업합니다.

```text
1. 샘플 메뉴 데이터를 준비한다.
2. 메뉴 카드 UI를 만든다.
3. 식당별로 메뉴를 묶어서 보여준다.
4. 날짜 선택 영역을 만든다.
5. 메뉴 카드를 누르면 상세 화면으로 이동하게 한다.
6. 메뉴 선택 체크박스를 표시한다.
7. 승헌 님과 체크 상태가 요약 화면으로 이어질 수 있게 맞춘다.
```

한 번에 전체 화면을 완성하려고 하지 말고, 위 순서대로 작은 단위로 진행합니다.

---

## 7. AI에게 요청할 때의 방향

AI에게는 한 번에 큰 화면 전체를 맡기지 말고, 작은 단위로 요청합니다.

좋은 작업 순서는 다음과 같습니다.

```text
1. 메뉴 카드 위젯 만들기
2. 식당별 메뉴 그룹 위젯 만들기
3. 오늘의 학식 화면에 그룹 목록 표시하기
4. 메뉴 상세 화면 이동 연결하기
5. 체크박스 UI 추가하기
```

AI에게 요청한 뒤에는 반드시 앱을 실행해서 화면이 잘 나오는지 확인합니다.

---

## 8. 승헌 님과 맞춰야 할 부분

준영 님의 화면에서 선택한 메뉴는 승헌 님의 요약 화면으로 이어져야 합니다.

따라서 아래 내용은 승헌 님과 함께 맞춥니다.

```text
- 메뉴를 선택했을 때 어떤 데이터가 저장되는지
- 선택한 메뉴를 요약 화면에서 어떻게 사용하는지
- 메뉴 카드와 요약 화면의 디자인 톤
- 요약 화면으로 이동하는 버튼 위치
```

---

## 9. 완료 기준

아래 조건을 만족하면 이번 작업이 완료된 것으로 봅니다.

```text
- 앱에서 오늘의 학식 화면이 보인다.
- 식당별로 메뉴가 묶여서 보인다.
- 메뉴 카드가 화면에 표시된다.
- 메뉴 카드를 누르면 상세 화면으로 이동할 수 있다.
- 체크박스를 누를 수 있다.
- 앱 실행 중 오류가 없다.
```

## 10. 커밋하고 PR 올리기

작업이 끝나면 커밋하고 GitHub에 올립니다.

```bash
git status
git add .
git commit -m "feat: add menu list screen"
git push origin feature/menu-list-screen
```

그다음 GitHub에서 Pull Request를 만들고 팀원 A에게 확인을 요청합니다.
